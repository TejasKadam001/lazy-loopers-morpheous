# Sero

Sero is an AI-powered clinical diagnostics engine designed for regulated institutions and healthcare providers. It features a multimodal diagnostic interface, real-time predictive bed turnover modeling, zero-latency clinical dashboards, and immutable audit trails via a secure blockchain layer.

This repository holds the raw HTML, CSS, assets, and frontend UI templates for the Sero landing pages, solutions dashboards, and knowledge hubs.

## Table of Contents
- `index.html` - The unified home page
- `solutions.html` - Detailed breakdown of Sero products
- `about.html` - The Sero founding team
- `wellbeing.html` - The knowledge and medical research hub
- `/css`, `/js`, `/images`, `/media` - Asset directories for styling and logic

## Features
- Sleek, custom dark-mode aesthetics
- GSAP and Lottie powered animations
- Modular HTML structure for easy refactoring

## Development
Run any local server to serve the static files:
```bash
python3 -m http.server 3000
```
Then navigate to `localhost:3000` in your browser.
